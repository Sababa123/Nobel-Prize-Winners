Nobel Prize Data Analysis & Predictive Modeling

Performed a comprehensive exploratory and predictive analysis on a 1,000‑record Nobel Prize laureate dataset. Identified key patterns in gender, birth country, age, and institutional affiliation through decade‑wise and geospatial visualizations. Built predictive models—including ARIMA for award‑age forecasting, linear regressions for institutional growth and economic correlates, and network‑based link prediction—to forecast trends and map collaboration networks. Delivered actionable insights and recommendations.
